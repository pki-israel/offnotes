Tipo de nota

Compartilhar nota

Este app não usa servidor: a nota inteira é codificada dentro do link. O link funciona para quem tiver acesso a esta mesma página. Notas muito grandes geram links longos.

Histórico de versões

Coleções

Coleções agrupam notas, como pastas. Excluir uma coleção não exclui as notas dela.

Lixeira

Backup

Suas notas ficam salvas apenas neste navegador (localStorage). Faça backups periódicos para não perder nada se o navegador for limpo.

A restauração mescla o backup com as notas atuais: notas com a mesma identificação são substituídas pela versão mais recente; as demais são adicionadas.
